ParseDatabase

Santiago Cano, Agustin Rico

The program writes a .csv file containing relevant information about proteins
in a specific format (Id, Name, Created Date, Formal Organism, Sequence). 
Proteins are taken from a recognized database.
 
In order to perform the objective of the program, an execution of it must be
done on the Command Console (ghc parseDatabase.hs), then do another call
inserting the name of the file to be readen (parseDatabase "psd7003.xml").
The program will create the .csv file containing the important information
the desired format.  

GHCi, version 8.6.3
